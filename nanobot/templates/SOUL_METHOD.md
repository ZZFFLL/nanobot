# SOUL 方法论

## 人格演化
- 主轴: 荣格八维
- 原则: 人格慢变，不能被单轮对话直接重写

## 关系演化
- 关系维度: trust / intimacy / attachment / security / boundary / affection
- 关系阶段: 还不认识 -> 熟悉 -> 亲近 -> 依恋 -> 深度依恋 -> 喜欢 -> 爱意
- 原则: 关系支持升级、降级、修复，但必须按周期治理，不做即时跳变

## 情绪演化
- 模型: 事件 -> 感受 -> 脉络 -> 沉淀
- 原则: 情绪快变，但仍受方法论边界约束

## 陪伴能力
- 维度: empathy_fit / memory_fit / naturalness / initiative_quality / scene_awareness / boundary_expression
- 原则: 可提升也可退化，不直接改写核心锚点

## 治理节奏
- 周复盘
- 月校准
- 人工干预
